# IPL 2024 Data Analysis (Python)

This project analyzes IPL 2024 match data using Python.
The dataset is explored to understand team performance, toss impact,
winning margins, and Player of the Match trends.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
- IPL_2024_dataset.csv
- Total matches: 75
- Total columns: 18

## Analysis Performed
- Dataset overview and cleaning
- Top Players of the Match analysis
- Wins by runs and wickets distribution
- Toss decision impact on match result
- Top winning teams
- Matches played in different cities
- Basic future performance prediction

## How to Run
1. Install required libraries  
   `pip install pandas numpy matplotlib seaborn`
2. Place `IPL_2024_dataset.csv` in the project folder
3. Run the Python file or notebook

## Output
- Bar charts
- Histograms
- Pie charts
- Insights based on IPL 2024 data

## Conclusion
Based on IPL 2024 data, **Chennai Super Kings** appears to be the most
successful team and may perform well in future seasons.
